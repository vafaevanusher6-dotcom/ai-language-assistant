# AI Language Assistant

This is my first AI project.

The goal of this project is to create an AI assistant
that helps students practice languages such as
Korean (TOPIK) and English (IELTS).

## Why I created this project
I am preparing for university in IT / AI and I am
personally learning Korean and English.
This project combines my interest in languages and AI.

## Current features
- Simple question and answer practice
- User input checking
- Basic feedback

## Future plans
- Add AI-based feedback
- Create a web or Telegram interface
- Improve language evaluation logic
